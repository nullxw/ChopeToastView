ChopeToastView
==============

iOS toast view


## Getting Started

#### CocoaPods

```ruby
platform :ios
pod "ChopeToastView"
```

## Usage

```objective-c
[[ChopeToastView ToastViewWithMessage:@"message"] show];
```
